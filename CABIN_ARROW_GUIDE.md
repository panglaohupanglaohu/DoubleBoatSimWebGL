# 舱室箭头引导功能 | Cabin Arrow Guide Feature

## ✨ 新功能 | New Feature

### 箭头指示器 | Arrow Indicator

在舱室入口处添加了**醒目的箭头指示器**，引导用户进入舱室。

## 🎯 功能特点 | Features

### 1. 双重箭头设计
- **主箭头**：绿色实心箭头，指向舱室入口
- **发光箭头**：半透明发光箭头，增强视觉效果

### 2. 动画效果
- **上下浮动**：箭头持续上下浮动（±0.3米）
- **发光脉冲**：发光箭头有脉冲效果（透明度和亮度变化）

### 3. 视觉特点
- **颜色**：绿色 (#00ff00)，与舱室标注一致
- **大小**：箭头长度3米，箭头头部1米
- **位置**：位于标记点上方，指向舱室入口中心

## 📍 位置说明 | Location

### 箭头位置
- **起点**：标记点上方3米
- **方向**：垂直向下指向舱室入口
- **终点**：舱室入口中心位置

### 视觉层次
```
     ↑ 文字标签
     ↑ 箭头指示器（上下浮动）
     ↑ 发光标记点
     ↑ 脉冲光环
     ↑ 高亮平面（舱室入口）
```

## 🎨 视觉效果 | Visual Effects

### 动画参数
- **浮动速度**：2秒一个周期
- **脉冲速度**：3秒一个周期
- **浮动幅度**：±0.3米
- **发光强度**：1.0 - 1.8（动态变化）

### 颜色方案
- **主箭头**：纯绿色，不透明
- **发光箭头**：绿色，半透明（40%-80%透明度）
- **发光强度**：高亮度（emissiveIntensity: 1.0-1.8）

## 🎮 使用方法 | Usage

### 步骤 1：查找箭头
在船体上查找：
- ⬇️ **绿色箭头**（上下浮动）
- 📍 **箭头指向舱室入口**

### 步骤 2：跟随箭头
- 箭头指向的位置就是**舱室入口**
- 箭头下方是**高亮平面**（可点击区域）

### 步骤 3：双击进入
- 将鼠标移到箭头指向的**高亮平面**区域
- **双击**进入舱室

## 📊 完整视觉元素 | Complete Visual Elements

现在每个舱室入口有**6种视觉元素**：

1. **⬇️ 箭头指示器** - 引导用户（新增）
2. **🟢 高亮平面** - 可点击区域
3. **⬜ 发光边框** - 舱室边界
4. **🔵 发光标记点** - 位置标记
5. **📝 文字标签** - 舱室名称
6. **💫 脉冲光环** - 装饰效果

## 🔧 技术实现 | Technical Implementation

### 箭头创建
```javascript
const arrowHelper = new THREE.ArrowHelper(
  new THREE.Vector3(0, -1, 0), // 向下指向
  arrowOrigin,                 // 起点位置
  3,                           // 箭头长度
  0x00ff00,                    // 绿色
  1,                           // 箭头头部长度
  0.8                          // 箭头头部宽度
);
```

### 动画更新
```javascript
// 上下浮动
const floatOffset = Math.sin(time * 2) * 0.3;
arrowHelper.position.y = arrowOrigin.y + floatOffset;

// 发光脉冲
const glowPulse = Math.sin(time * 3) * 0.5 + 0.5;
glowArrowHelper.material.opacity = 0.4 + glowPulse * 0.4;
```
