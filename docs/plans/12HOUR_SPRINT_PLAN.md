### 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端 3D** | Three.js r128+ | WebGL 渲染，复用 DoubleBoatSimWebGL |
| **前端 UI** | React 18 + Tailwind | 响应式仪表盘 |
| **后端 API** | Python FastAPI | 高性能 REST API |
| **实时通信** | WebSocket | <100ms 延迟推送 |
| **时序数据** | SQLite (轻量) / InfluxDB | 历史数据存储 |
| **仿真数据** | 公开 AIS API + 自研模拟器 | 测试数据源 |
| **AI 集成** | OpenClaw Agent | 自然语言交互 |
