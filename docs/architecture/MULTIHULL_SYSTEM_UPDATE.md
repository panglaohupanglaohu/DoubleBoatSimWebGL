## 📋 多体船特性分析

### 多体船类型

| 类型 | 特点 | 应用场景 |
|------|------|----------|
| **双体船 (Catamaran)** | 2 个片体 + 连接桥 | 客滚船/科考船/游艇 |
| **三体船 (Trimaran)** | 3 个片体 (主船体 +2 侧体) | 高性能军舰/豪华游艇 |

### 多体船特殊需求

#### 1. 结构特点

**双体船**:
- ✅ 2 个对称片体
- ✅ 连接桥结构
- ✅ 双机双桨
- ✅ 稳定性高

**三体船**:
- ✅ 1 个主船体 + 2 个侧体
- ✅ 复杂的连接结构
- ✅ 可能 3 机或 1 机 + 侧体辅助推进
- ✅ 稳定性更高，阻力更低

#### 2. 监控需求差异

| 系统 | 单体船 | 双体船 | 三体船 |
|------|--------|--------|--------|
| **主机监控** | 1 台主机 | 2 台主机 | 1-3 台主机 |
| **电力管理** | 1-2 发电机 | 2-4 发电机 | 2-6 发电机 |
| **燃油系统** | 单系统 | 双系统 (左右独立) | 三系统 (主 + 侧体) |
| **稳性监控** | GM 值 | 双体 GM+ 连接桥应力 | 三体 GM+ 侧体载荷 |
| **结构监测** | 单船体 | 连接桥应力 | 主侧体连接处应力 |

## 🔧 新增功能模块

### 1. 多体船配置管理

**文件**: `multihull_config.py`

```python
from enum import Enum
from dataclasses import dataclass

class HullType(Enum):
    CATAMARAN = "catamaran"  # 双体船
    TRIMARAN = "trimaran"    # 三体船

@dataclass
class MultihullConfig:
    hull_type: HullType
    number_of_hulls: int  # 2 or 3
    number_of_engines: int  # 1-6
    bridge_structure: bool  # 是否有连接桥
    
    # 双体船参数
    demihull_spacing: float  # 片体间距 (米)
    
    # 三体船参数
    side_hull_size_ratio: float  # 侧体大小比例 (相对于主船体)
    side_hull_position: float  # 侧体位置 (距船中距离)
```

## 📊 多体船特殊监控点

### 双体船监控点

| 系统 | 监控点 | 传感器 | 报警阈值 |
|------|--------|--------|----------|
| **连接桥** | 应力 | 应变片 | <200 MPa |
| **左片体** | RPM/温度/压力 | 主机传感器 | 同单体船 |
| **右片体** | RPM/温度/压力 | 主机传感器 | 同单体船 |
| **燃油** | 左右舱液位 | 液位传感器 | 偏差<10% |
| **稳性** | GM 值 | 姿态传感器 | GM>1.5m |

### 三体船监控点

| 系统 | 监控点 | 传感器 | 报警阈值 |
|------|--------|--------|----------|
| **主船体** | RPM/温度/压力 | 主机传感器 | 同单体船 |
| **左侧体** | RPM/温度/压力 (如有主机) | 主机传感器 | 同单体船 |
| **右侧体** | RPM/温度/压力 (如有主机) | 主机传感器 | 同单体船 |
| **连接结构** | 主 - 侧体连接处应力 | 应变片 | <250 MPa |
| **侧体载荷** | 出水风险 | 姿态 + 液位 | 横倾<15° |
