## 核心知识点

### 1. 现代船舶工程趋势
- 更注重成本效益运营
- 燃油效率更高的发动机
- 减少港口停留时间
- 提高设备可靠性
- 更少工程师，更多自动化操作

### 2. 船舶工程师核心职责
- 理解所有船舶机械的工作原理、结构和操作
- 遵循正确和安全的操作程序
- 遵守环保法规（油污排放、污水、黑烟）
- 环境意识（废气排放、环保制冷剂、灭火剂）

### 3. 适用证书级别
- Class 4 和 Class 3 工程师资格证书
- 工程师培训计划前两年
- 船长证书的工程知识大纲
