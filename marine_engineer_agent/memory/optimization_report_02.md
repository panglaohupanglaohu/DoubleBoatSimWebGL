### 2. 代码修复

#### fault_diagnosis.py - 停用词过滤边界情况

**问题描述：**
```python
# 输入："的是有的"
# 期望输出：[]
# 实际输出：["的是有的"]  ❌
```

**原因分析：**
- 原代码仅检查整个 token 是否在停用词集合中
- "的是有的" 不在停用词集合中（集合只包含单个词）
- 但所有字符都是停用词字符

**修复方案：**
```python
# 新增字符级停用词检测
stop_chars = set("".join(stop_words))  # {'的', '是', '有', ...}

for p in parts:
    if not p or p in stop_words or len(p) <= 1:
        continue
    # 检查是否所有字符都是停用词字符
    if all(c in stop_chars for c in p):
        continue
    keywords.append(p)
```

**验证结果：**
- ✅ `test_only_stop_words` 测试通过
- ✅ 不影响正常关键词提取
