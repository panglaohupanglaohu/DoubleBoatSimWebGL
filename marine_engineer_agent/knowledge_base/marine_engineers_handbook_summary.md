## 概述

### 手册定位

《Marine Engineers' Handbook》是船舶轮机工程领域的经典参考书，由 John Madison Labberton 编写，首次出版于 1945 年。该手册为船舶工程师提供：

- 全面的轮机工程知识
- 实用的工程计算公式和表格
- 设备操作和维护规范
- 故障诊断方法

### 适用范围

- 商船轮机工程师
- 海军舰艇工程人员
- 船舶维修技术人员
- 海事院校师生

## 蒸汽动力系统

### 锅炉类型

| 类型 | 特点 | 应用 |
|------|------|------|
| 水管锅炉 | 效率高，压力高 | 大型船舶 |
| 火管锅炉 | 结构简单，维护方便 | 小型船舶 |
| 组合锅炉 | 灵活性高 | 多用途船舶 |

### 锅炉主要部件

1. **炉膛**
   - 燃料燃烧空间
   - 耐火材料衬里
   - 燃烧器安装位置

2. **水冷壁**
   - 吸收辐射热量
   - 保护炉墙
   - 产生蒸汽

3. **过热器**
   - 提高蒸汽温度
   - 增加热效率
   - 防止汽轮机叶片腐蚀

4. **省煤器**
   - 预热给水
   - 回收烟气余热
   - 提高锅炉效率

5. **空气预热器**
   - 加热燃烧空气
   - 改善燃烧效率
   - 降低排烟温度

### 锅炉运行参数

| 参数 | 典型值 | 单位 |
|------|--------|------|
| 工作压力 | 15-60 | bar |
| 蒸汽温度 | 200-450 | °C |
| 蒸发量 | 1-100 | t/h |
| 热效率 | 75-90 | % |

### 锅炉维护要点

- 定期清洗水侧和火侧
- 检查耐火材料状况
- 校验安全阀
- 监测水质指标
- 记录运行参数

## 内燃机

### 柴油机分类

| 分类标准 | 类型 | 特点 |
|----------|------|------|
| 工作循环 | 四冲程 | 效率高，转速高 |
| | 二冲程 | 功率密度大，低速 |
| 转速 | 低速 (<300rpm) | 直接驱动螺旋桨 |
| | 中速 (300-1000rpm) | 需齿轮箱 |
| | 高速 (>1000rpm) | 小型船舶 |
| 增压方式 | 机械增压 | 响应快 |
| | 涡轮增压 | 效率高 |

### 柴油机主要系统

1. **燃油系统**
   - 燃油储存和输送
   - 燃油净化 (分油机)
   - 高压燃油喷射
   - 喷油定时控制

2. **润滑系统**
   - 主润滑油循环
   - 气缸油润滑
   - 润滑油冷却和净化
   - 油压和油温监控

3. **冷却系统**
   - 高温淡水冷却 (缸套)
   - 低温淡水冷却 (增压空气)
   - 海水冷却
   - 膨胀水箱

4. **启动系统**
   - 压缩空气启动
   - 电动启动 (小型机)
   - 启动空气瓶
   - 启动阀和分配器

5. **进排气系统**
   - 空气滤清器
   - 涡轮增压器
   - 中冷器
   - 排气消声器

### 柴油机性能参数

| 参数 | 符号 | 单位 | 说明 |
|------|------|------|------|
| 指示功率 | Pi | kW | 气缸内产生的功率 |
| 制动功率 | Pb | kW | 输出轴功率 |
| 机械效率 | ηm | % | Pb/Pi |
| 燃油消耗率 | SFC | g/kWh | 单位功率燃油消耗 |
| 平均有效压力 | MEP | bar | 做功能力指标 |

### 典型 SFC 值

| 发动机类型 | SFC 范围 | 备注 |
|------------|----------|------|
| 低速二冲程 | 160-175 | 效率最高 |
| 中速四冲程 | 175-190 | 常用辅机 |
| 高速四冲程 | 190-220 | 小型船舶 |

## 涡轮机

### 蒸汽涡轮机

**工作原理：** 高压蒸汽通过喷嘴膨胀，推动叶片旋转

**类型：**
- 冲动式涡轮机
- 反动式涡轮机
- 组合式涡轮机

**应用：**
- 大型船舶主推进
- 液化天然气 (LNG) 船
- 航空母舰

### 燃气涡轮机

**工作原理：** 压缩空气与燃料燃烧，高温燃气推动涡轮

**特点：**
- 功率密度高
- 启动快
- 噪音低
- 燃油消耗率高

**应用：**
- 军舰
- 高速客船
- 联合动力系统

## 工程计算

### 功率计算

**指示功率：**
```
Pi = (Pm × L × A × N × K) / 60

其中:
Pm = 平均指示压力 (bar)
L = 冲程 (m)
A = 活塞面积 (m²)
N = 转速 (rpm)
K = 气缸数
```

**制动功率：**
```
Pb = (2π × N × T) / 60000

其中:
N = 转速 (rpm)
T = 扭矩 (N·m)
```

### 燃油消耗计算

**小时燃油消耗：**
```
F = Pb × SFC / 1000

其中:
F = 燃油消耗量 (kg/h)
Pb = 制动功率 (kW)
SFC = 燃油消耗率 (g/kWh)
```

**航次燃油消耗：**
```
Fvoyage = F × t / 1000

其中:
t = 航行时间 (h)
```

### 热效率计算

**制动热效率：**
```
ηb = 3600 / (SFC × Qnet)

其中:
Qnet = 燃油低位热值 (kJ/kg)
```

**典型热效率值：**
- 低速柴油机：45-50%
- 中速柴油机：40-45%
- 高速柴油机：35-40%
- 蒸汽涡轮机：25-35%
- 燃气涡轮机：25-30%

### 常用单位换算

| 换算 | 公式 |
|------|------|
| 马力 → 千瓦 | 1 HP = 0.7457 kW |
| 千瓦 → 马力 | 1 kW = 1.341 HP |
| 吨 → 磅 | 1 t = 2204.62 lb |
| 海里 → 公里 | 1 nm = 1.852 km |
| 节 → m/s | 1 kn = 0.5144 m/s |
| PSI → bar | 1 PSI = 0.06895 bar |

## 维护规范

### 日常检查

| 项目 | 频率 | 内容 |
|------|------|------|
| 润滑油位 | 每班 | 检查油位，必要时添加 |
| 冷却水位 | 每班 | 检查膨胀水箱水位 |
| 启动空气压力 | 每班 | 确保压力正常 |
| 燃油温度 | 每班 | 检查加热温度 |
| 排气温度 | 每班 | 记录各缸排温 |
| 润滑油压力 | 每班 | 检查主油道压力 |

### 定期维护

| 周期 | 项目 |
|------|------|
| 每周 | 测试应急设备，检查安全阀 |
| 每月 | 清洗滤器，检查皮带张力 |
| 每 3 月 | 润滑油取样分析，校验仪表 |
| 每 6 月 | 清洗冷却器，检查轴承 |
| 每年 | 吊缸检查，校验安全系统 |
| 每 2-5 年 | 大修，更换磨损件 |

### 润滑油管理

**关键指标：**
- 粘度变化：< ±20%
- 总碱值 (TBN)：> 50% 新油值
- 水分：< 0.2%
- 闪点：> 150°C

**维护措施：**
- 定期取样分析
- 使用分油机净化
- 及时补充新油
- 避免不同品牌混用

## 安全规范

### 个人防护装备 (PPE)

- 安全帽
- 防护眼镜
- 耳塞/耳罩
- 防护手套
- 安全鞋
- 工作服

### 锁闭/挂牌 (LOTO)

**程序：**
1. 识别能源类型
2. 通知相关人员
3. 关闭设备
4. 隔离能源
5. 上锁挂牌
6. 验证隔离
7. 进行维修
8. 拆除锁具
9. 测试设备
10. 通知恢复运行

### 密闭空间进入

**安全要求：**
- 气体检测 (氧气、可燃气体、有毒气体)
- 通风换气
- 专人监护
- 应急设备准备
- 进入许可

### 热工作业

**预防措施：**
- 清除可燃物
- 配备灭火器材
- 设置防火毯
- 气体检测
- 作业许可

### 高压系统作业

**注意事项：**
- 泄压后作业
- 使用专用工具
- 穿戴防护装备
- 设置警示标志

## 故障诊断

### 柴油机常见故障

| 故障现象 | 可能原因 | 处理方法 |
|----------|----------|----------|
| 启动困难 | 启动空气压力低、燃油系统故障 | 检查空气瓶、燃油供应 |
| 功率不足 | 燃油喷射不良、增压器故障 | 检查喷油器、增压器 |
| 排气温度高 | 超负荷、喷油定时晚 | 减负荷、调整定时 |
| 润滑油压力低 | 油泵故障、轴承磨损 | 检查油泵、轴承间隙 |
| 冷却水温高 | 水泵故障、冷却器脏堵 | 检查水泵、清洗冷却器 |
| 异常振动 | 对中不良、轴承损坏 | 检查对中、更换轴承 |

### 锅炉常见故障

| 故障现象 | 可能原因 | 处理方法 |
|----------|----------|----------|
| 汽压低 | 燃烧不良、负荷过大 | 调整燃烧、减负荷 |
| 水位异常 | 给水系统故障、水位计故障 | 检查给水泵、校验水位计 |
| 排烟温度高 | 受热面积灰、燃烧不良 | 吹灰、调整燃烧 |
| 安全阀起跳 | 超压、安全阀故障 | 检查压力控制、校验安全阀 |
