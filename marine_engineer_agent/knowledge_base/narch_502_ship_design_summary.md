## 🔑 关键知识点

### 1. 船舶设计流程

```
概念设计 → 合同设计 → 详细设计 → 生产设计
   ↓           ↓           ↓           ↓
可行性     技术规格    施工图纸    制造指令
```

### 2. 设计参数

| 参数类别 | 主要内容 |
|----------|----------|
| 主尺度 | 总长、垂线间长、型宽、型深、吃水 |
| 船型系数 | 方形系数、棱形系数、水线面系数 |
| 性能 | 航速、续航力、自持力 |
| 重量 | 空船重量、载重量、排水量 |

### 3. 水动力学基础

**阻力组成：**
- 摩擦阻力 (Frictional Resistance)
- 兴波阻力 (Wave Making Resistance)
- 剩余阻力 (Residual Resistance)
- 空气阻力 (Air Resistance)

**推进效率：**
```
总效率 = 船身效率 × 螺旋桨效率 × 相对旋转效率 × 轴系效率
```

### 4. 稳性要求

**初稳性：**
- GM > 0 (正值)
- GM 值根据船型和使用条件确定

**大倾角稳性：**
- 复原力臂曲线 (GZ Curve)
- 稳性衡准数
