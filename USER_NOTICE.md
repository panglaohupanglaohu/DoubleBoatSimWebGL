## 🎮 如何使用 | How to Use

### 第一步：刷新页面
```
按 Ctrl + Shift + R（强制刷新）
```

### 第二步：等待初始化（2-3秒）
打开F12控制台，应该看到：
```
✅ Cabin system initialized
✅ Realtime display system initialized
```

### 第三步：查找舱室标记

**你应该能看到**（非常明显）：
- 🟢 **两个绿色发光球**（一闪一闪的）
- 📝 **"零部件仓库"** 和 **"数据中心"** 标签
- ⬜ 绿色线框盒子（在旋转）

### 第四步：进入舱室

**方式 1：点击**
1. 点击绿色发光球
2. 或点击舱室标签
3. 相机会平滑飞入舱室

**方式 2：GUI菜单**
1. 打开右上角GUI
2. 找到 "舱室系统 | Cabin System"
3. 下拉菜单选择舱室

**方式 3：控制台**
```javascript
cabinManager.enterCabin('parts-warehouse')  // 零部件仓库
cabinManager.enterCabin('data-center')      // 数据中心
```

### 第五步：退出舱室

**方式 1：ESC键** ⭐ 推荐
```
按 ESC 键
```

**方式 2：GUI菜单**
```
点击"退出舱室"按钮
```

**方式 3：控制台**
```javascript
cabinManager.exitCabin()
```

### 第六步：查看实时数据

**环绕相机360度**，你会看到：
- ⛽ **左前方**：燃油表盘（3D仪表）
- 🏗️ **右中部**：吊机状态灯（绿色发光）
- 👥 **中部**：人员标记（3个小人）
- 🧪 **左后方**：实验进度条
- 📦 **右后方**：仓储指示器
- 🌬️ **上方**：风向箭头（大箭头）

## 🐛 如果还是看不到... | Troubleshooting

### 1. 检查控制台
按F12，查看是否有错误消息

### 2. 等待更久
有时初始化需要3-5秒

### 3. 使用调试命令
在控制台输入：

```javascript
// 检查舱室系统
console.log(cabinManager);
console.log(cabinManager.getCabinsInfo());

// 检查数据系统
console.log(realtimeDisplaySystem);
realtimeDisplaySystem.displayObjects.forEach((display, key) => {
  console.log(key, display.group.position);
});

// 手动进入舱室测试
cabinManager.enterCabin('parts-warehouse');

// 移动相机到数据对象
camera.position.set(-15, 8, -15);  // 燃油表盘
camera.lookAt(-15, 8, -20);
```

### 4. 清除浏览器缓存
```
Ctrl + Shift + Delete
选择"缓存的图片和文件"
清除
```

### 5. 更换浏览器
推荐使用：
- Chrome
- Edge
- Firefox
