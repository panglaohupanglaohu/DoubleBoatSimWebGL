## 🔔 提醒机制

由于 cron 不支持 main session，使用以下提醒方式：

### 1. 文件提醒
```bash
# 每小时检查时间
while true; do
  hour=$(date +%H)
  minute=$(date +%M)
  
  if [ "$minute" = "00" ]; then
    echo "⏰ $hour:00 状态报告时间到！" >> /tmp/status_reminder.log
  fi
  
  sleep 60
done
```
