# 船身稳定分析器 | Ship Stability Analyzer

## 📋 概述 | Overview

船身稳定分析器是一个智能系统，能够自动分析船体状态并给出参数调整建议，帮助船体快速恢复稳定。

**功能 | Features**:
- ✅ 实时分析船体稳定性
- ✅ 自动检测问题（倾斜、下沉、振荡等）
- ✅ 智能参数调整建议
- ✅ 一键稳定化功能
- ✅ 详细的调整报告

## 🚀 使用方法 | How to Use

### 1. 访问重构版页面

```
http://localhost:3000/index-refactored.html
```

### 2. 使用稳定化功能

1. **观察船体状态**
   - 左下角状态面板会显示当前稳定状态
   - ✅ 稳定 | Stable - 船体正常
   - ⚠️ 警告 - 有轻微问题
   - ❌ 不稳定 - 需要调整

2. **点击稳定化按钮**
   - 在 GUI 控制面板中找到 **"⚓ 浮力与稳定性 | Buoyancy & Stability"** 文件夹
   - 在 **"🔄 重置船体 | Reset Boat"** 按钮下方
   - 点击 **"⚖️ 船身稳定 | Stabilize Ship"** 按钮

3. **查看结果**
   - 状态面板会显示详细的稳定化结果
   - 包括：问题诊断、已应用的调整、参数建议
   - 控制台也会输出详细信息

## 🔍 分析功能 | Analysis Features

### 检测的问题 | Detected Issues

| 问题 | 阈值 | 说明 |
|------|------|------|
| 严重倾斜 | > 60° | 有倾覆风险 |
| 过度下沉 | > 10m | 船体下沉过深 |
| 较大倾斜 | > 30° | 需要关注 |
| 较深下沉 | > 7m | 下沉较深 |
| 角速度过大 | > 2.0 rad/s | 船体振荡 |
| 线速度过大 | > 5.0 m/s | 移动过快 |
| 漂移过大 | > 100m | 偏离位置过远 |

### 自动调整参数 | Auto-Adjustments

系统会根据检测到的问题自动调整以下参数：

1. **浮力系数 | Buoyancy Coefficient**
   - 问题：下沉过深
   - 调整：增加浮力系数（400 → 600-800）

2. **阻尼系数 | Drag Coefficient**
   - 问题：振荡过大
   - 调整：增加阻尼（8 → 12-20）

3. **自稳刚度 | Stabilizer Stiffness**
   - 问题：倾斜过大
   - 调整：增加刚度（12 → 15-20）

4. **摇晃增强 | Wobble Boost**
   - 问题：摇晃过大
   - 调整：减小系数（0.8 → 0.5-0.6）

5. **船体质量 | Boat Mass**
   - 问题：质量过轻导致不稳定
   - 调整：增加质量（7000 → 9000-12000 kg）

- 快速恢复稳定状态

## 🔧 技术细节 | Technical Details

### 分析算法 | Analysis Algorithm

1. **指标计算 | Metrics Calculation**
   ```javascript
   - 倾斜角度：船体上方向与世界上方向的夹角
   - 下沉深度：船体位置与水面高度的差值
   - 角速度：船体旋转速度
   - 线速度：船体移动速度
   - 漂移距离：船体偏离原点的距离
   ```

2. **问题检测 | Issue Detection**
   ```javascript
   if (tilt > 60°) → 严重倾斜
   if (sink > 10m) → 过度下沉
   if (angularSpeed > 2.0) → 振荡过大
   ```

3. **参数调整 | Parameter Adjustment**
   ```javascript
   // 根据问题严重程度计算调整值
   adjustment = baseValue + (problemSeverity - threshold) * factor
   ```

### 调整策略 | Adjustment Strategy

| 问题 | 调整参数 | 计算公式 |
|------|----------|----------|
| 倾斜 > 20° | 自稳刚度 | stiffness + (tilt - 20) * 0.3 |
| 下沉 > 5m | 浮力系数 | buoyancy + (sink - 5) * 50 |
| 角速度 > 1.0 | 阻尼系数 | drag + (angularSpeed - 1.0) * 3 |
| 质量过轻 | 船体质量 | mass + 2000 (max 20000) |

## 💡 最佳实践 | Best Practices

### 1. 何时使用稳定化

✅ **适合使用**:
- 船体明显不稳定时
- 参数调整后需要快速验证
- 极端天气后恢复
- 测试不同配置时

❌ **不需要使用**:
- 船体已经稳定
- 想要观察自然物理效果
- 测试极端条件下的表现

### 2. 参数调整建议

**保守调整 | Conservative**:
- 每次调整幅度较小
- 逐步优化
- 观察效果后再继续

**激进调整 | Aggressive**:
- 一次性大幅调整
- 快速达到稳定
- 可能失去真实感

### 3. 结合手动调整

1. 先使用稳定化功能快速恢复
2. 查看建议的参数值
3. 在GUI中手动微调
4. 找到最佳平衡点

## 🐛 故障排除 | Troubleshooting

### 问题1：稳定化后仍然不稳定

**可能原因 | Possible Causes**:
- 参数调整幅度不够
- 船体质量过轻
- 极端天气条件

**解决方案 | Solutions**:
- 再次点击稳定化按钮
- 手动增加质量到 10000-15000 kg
- 降低天气强度

### 问题2：稳定化后船体过于僵硬

**可能原因 | Possible Causes**:
- 自稳刚度设置过高
- 阻尼系数过大
- 摇晃增强系数过小

**解决方案 | Solutions**:
- 手动减小自稳刚度（15 → 12）
- 减小阻尼系数（12 → 8）
- 增加摇晃增强（0.5 → 0.8）

### 问题3：稳定化按钮无响应

**检查 | Check**:
- 船体是否已加载
- 浏览器控制台是否有错误
- 稳定分析器是否初始化

**解决 | Solution**:
- 等待船体加载完成
- 刷新页面
- 检查控制台错误信息
